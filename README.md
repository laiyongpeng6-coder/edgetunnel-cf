# edgetunnel-cf

Multi-user subscription management proxy based on cmliu/edgetunnel.

## Features
- vless ws/tls proxy
- Multi-user subscription links
- Monthly bandwidth quota
- Expiration dates
- Enable/disable users
- Traffic tracking
- Subscription-Userinfo header
- Admin dashboard
- User status page

## Deployment
Auto-deploys via Cloudflare Pages from this repo.
